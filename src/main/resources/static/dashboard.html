How this works: LibreLog does not duplicate show templates or smart blocks from LibreTime. It lists show instances that exist on the calendar day you picked (what actually airs). You place slots for spots or library tracks for trafficking and reconciliation. Use Clock schedule below to map station-local hours to clock templates, then Apply clock schedule so each show instance picks the right clock by its start time. Use Apply clock on a single show when you want to override with one template. Use Preload from LibreTime to copy today’s already-scheduled files into slots as tracks. Smart blocks fill inside LibreTime at play time — Preload only picks concrete library files already on the schedule.
Times: the date picker uses station local time (see clock above). LibreTime's UI may show times in its own configured timezone (often UTC) — if your shows look offset (e.g. an "8 AM" show landing at 1 AM station time), update general.timezone in LibreTime's config.yml and restart it.

Carts are rotating bins. A music cart picks one library file each time it's used in a slot; a commercial cart picks one spot (and only approved spots are eligible). Push time uses the cart's separation policy (artist/title/sponsor/product cooldowns) to avoid back-to-back repeats, plus a freshness strategy: Rotation for music and commercials, or Newest first with a max age for news and voicetracks so stale audio never airs. Order-backed commercial carts re-sync automatically when the order's spots change. Use Apply clock in the Day Builder to drop carts into a show in one click.

What a clock is: a reusable pattern for one hour — an ordered list of slots (music cart, commercial cart, fixed track, spot, etc.). In Day Builder you map hours to clocks in the clock schedule, then apply that schedule to shows (or apply one clock to a single show).

After you click “New clock”: the clock only has a name until you add slots and click Save slots.

  1. Select the clock in the list on the left (click its name).
  2. Add slot for each piece of content in order (first row = first thing in the hour).
  3. In each row, set kind:
    • music cart — pick a library (music) cart from the Carts tab; at push time one song/image is chosen from that cart.
    • commercial cart — pick a commercial cart (often tied to an order); at push time one spot is chosen.
    • track — always the same LibreTime file: enter its numeric file id (find it under LibreTime Library).
    • spot — one specific spot by UUID (from Orders; power-user).
    • note — placeholder / timing filler (optional label + length).
  4. Optional: label and len s (default length in seconds) help timing before push.
  5. Click Save slots — the template is not stored until you do.

Prerequisite: build your carts in the Carts tab first so they appear in the cart dropdown. Named day parts for spot windows live under Station in Setup.
