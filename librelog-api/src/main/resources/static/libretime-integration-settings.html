Enter your JWT authentication token from LibreTime. The token will be encrypted before storage.
How to get your JWT token

Method 1: Via LibreTime Web UI

  1. Log in to your LibreTime web interface
  2. Navigate to Settings → API or User Settings → API Tokens
  3. Click Generate New Token or copy an existing token
  4. Copy the token (it's a long string, typically starting with "eyJ")
  5. Paste it in the field above

Method 2: Via LibreTime API (if you have admin access)

You can generate a token by making a POST request to:

POST {your-libretime-url}/api/v2/login
Body: {"username": "your-username", "password": "your-password"}

The response will include a JWT token in the token field.

Note: If you can't find the API token section in LibreTime, check your LibreTime version and documentation. Some versions may use API keys instead of JWT tokens. Contact your system administrator if you need assistance.
