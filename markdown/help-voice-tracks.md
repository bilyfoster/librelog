# Voice Tracks - Help for Beginners

> **What is a Voice Track?**  
> A voice track is a pre-recorded segment where a DJ or announcer speaks. It makes your station sound live even when no one is in the studio.

---

## What Are Voice Tracks Used For?

### Common Uses

| Type | Purpose | Example |
|------|---------|---------|
| **Show Intro** | Start of a show | "Good morning! You're listening to the Wake Up Show on KLOV!" |
| **Show Outro** | End of a show | "That's it for today. Join us tomorrow at the same time!" |
| **Song Intro** | Before a song | "Here's the latest from Taylor Swift..." |
| **Song Outro** | After a song | "That was 'Shake It Off' on your number one hit music station!" |
| **Liner** | Station branding | "KLOV 101.5 - Today's Best Music!" |
| **Teaser** | Preview upcoming content | "Stay tuned - we'll have your weather forecast in 5 minutes!" |
| **Transition** | Between show segments | "Now let's check in with traffic..." |

## Creating a Voice Track

### Step 1: Write the Script

1. Go to **Voice Tracks** → **New Voice Track**
2. Enter:
   - **Title**: Something descriptive (e.g., "Monday Morning Show Intro")
   - **Show Name**: Which show this is for
   - **Segment Type**: Choose from dropdown
   - **Scheduled Date**: When should this play?
   - **Scheduled Time**: What time? (e.g., "06:00:00" for 6 AM)
   - **Script Text**: Write exactly what to say

### Writing Tips

**Keep it Short:**
- Liner: 5-10 seconds
- Intro/Outro: 15-30 seconds
- Song intro: 10-15 seconds

**Use This Formula:**
```
[Greeting] + [Station Name] + [What's Coming Up] + [Excitement!]

Example:
"Good morning! You're listening to KLOV 101.5 FM. 
Coming up - we've got your traffic and weather, plus the Top 5 at 8. 
Let's kick things off with today's hottest hit!"
```

**Be Conversational:**
- ✅ "Hey, how's it going?"
- ❌ "You are currently listening to..."

---

### Step 2: Schedule It

Voice tracks play at a specific date and time. This lets you:
- Record shows days in advance
- Schedule intros for specific songs
- Plan holiday programming

**Example Schedule:**
- Monday 6:00 AM: Morning Show Intro
- Monday 6:15 AM: First song intro
- Monday 6:30 AM: Station liner
- Monday 10:00 AM: Mid-morning transition

---

### Step 3: Record the Audio

**Option A: Record in LibreLog**
(Some systems have built-in recording)

**Option B: Use External Software**
1. Record in Audacity, Adobe Audition, etc.
2. Export as MP3 or WAV
3. Save with descriptive filename:
   - `monday-morning-intro-2024-03-15.mp3`
   - `taylor-swift-intro-shake-it-off.mp3`

**Recording Tips:**
- Use a good microphone
- Record in a quiet room
- Keep consistent volume
- Leave 1-second silence at start and end
- Don't peak into the red (avoid distortion)

---

### Step 4: Upload the File

1. Open your voice track
2. Click **Upload Audio**
3. Select your recorded file
4. System will show:
   - File name
   - File size
   - Duration (in seconds)

---

### Step 5: Mark Status

Track your progress:

| Status | Meaning | Who Uses |
|--------|---------|----------|
| **DRAFT** | Script written, not recorded | Announcer/DJ |
| **RECORDED** | Audio uploaded | Technical producer |
| **APPROVED** | Quality checked, ready to air | Program director |
| **SCHEDULED** | In the daily log | System |
| **AIRED** | Already played | System |

## Managing Voice Tracks

### View Upcoming Tracks

1. Go to **Voice Tracks** → **Upcoming**
2. See everything scheduled for today and future
3. Check for any missing recordings

### Find Tracks by Show

Filter by show name to see all voice tracks for a specific program.

### Update a Track

Need to re-record? Change the script?
1. Find the voice track
2. Edit the text
3. Upload new audio file
4. Change status back to DRAFT or RECORDED

## Best Practices

### Do's ✅
- **Name files clearly**: Include date and purpose
- **Keep backup copies**: Don't delete original recordings
- **Record ahead**: Do a week's worth on Friday
- **Match the energy**: Morning tracks = energetic, overnight = mellow
- **Say the station name**: Reinforce branding
- **Time it right**: Practice reading to hit target duration

### Don'ts ❌
- **Don't rush**: Record when you have time to do it well
- **Don't eat/drink while recording**: Avoid mouth noises
- **Don't wing it**: Write scripts, don't improvise
- **Don't forget breath sounds**: Edit out heavy breathing
- **Don't overlap**: Keep voice tracks separate from songs

## Timing Guide

### How Long Should Voice Tracks Be?

| Type | Target Duration | Word Count |
|------|----------------|------------|
| Liner | 5-10 seconds | 10-20 words |
| Song Intro | 10-15 seconds | 20-30 words |
| Song Outro | 5-10 seconds | 10-20 words |
| Show Intro | 20-30 seconds | 40-60 words |
| Show Outro | 15-20 seconds | 30-40 words |
| Teaser | 10-15 seconds | 20-30 words |

### Speaking Pace

**Normal conversational speed**: ~150 words per minute

**Radio pace**: Slightly slower, ~130 words per minute

**To calculate**:  
- 30 seconds = about 65 words
- 15 seconds = about 30 words
- 10 seconds = about 20 words

## Troubleshooting

### Problem: "Voice track didn't play"
**Check**:
- Status is APPROVED or SCHEDULED
- Audio file uploaded correctly
- Scheduled date and time are correct
- In the daily log

### Problem: "Audio file too quiet/loud"
**Solution**:
- Re-record with proper levels
- Or use audio editing software to normalize
- Target: -16 LUFS for broadcast

### Problem: "File won't upload"
**Check**:
- File format (MP3 or WAV recommended)
- File size (keep under 50MB)
- Internet connection

### Problem: "Wrong duration showing"
**Solution**:
- Re-upload the file
- System calculates from actual audio file

## FAQ

**Q: Can I use voice tracks for live shows?**  
A: Yes! Many live shows use voice tracks for liners and transitions between segments.

**Q: How far ahead should I record?**  
A: At least a week. Some stations record a month ahead for regular features.

**Q: Can I re-use voice tracks?**  
A: Liners yes, time-specific intros no. Don't say "Good morning" in the afternoon!

**Q: What if the announcer is sick?**  
A: Voice tracks let shows continue even when the host isn't there. Just play pre-recorded content.

**Q: Do voice tracks replace live announcers?**  
A: They supplement them. Best stations use a mix of live and tracked content.

## Recording Equipment Tips

### Minimum Setup
- USB microphone ($50-100)
- Quiet room
- Pop filter (or make one with pantyhose and a hanger)
- Recording software (Audacity is free)

### Better Setup
- XLR microphone ($200+)
- Audio interface
- Acoustic treatment (foam panels)
- Professional headphones
- Boom arm for mic

### Recording Settings
- Sample rate: 44.1 kHz or 48 kHz
- Bit depth: 16-bit or 24-bit
- Export as: MP3 (192 kbps minimum) or WAV
