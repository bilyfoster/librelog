### ğŸ¤ Submitting Music

#### **Using the Enhanced Submission Form**

1. **Start Submission**
   - Go to https://music.gayphx.com/submit-new
   - Click "Submit Music" in the main navigation

2. **Select Your Artist**
   - Choose from your existing artist profiles
   - Artist information will automatically fill in
   - If you need a new artist, click "Add New Artist"

3. **Fill Track Information**
   - **Song Title**: Name of your track
   - **Genre**: Musical genre (optional)
   - **Audio File**: Upload your track (MP3, WAV, M4A, FLAC)

4. **Set Permissions**
   - **ISRC Request**: Check if you want an official ISRC code
   - **Radio Permission**: Allow radio play
   - **Public Display**: Show in public gallery
   - **Podcast Permission**: Allow podcast use
   - **Commercial Use**: Allow commercial use
   - **Rights Attestation**: Confirm you have the rights

5. **Professional Information** (Optional)
   - **PRO Affiliation**: ASCAP, BMI, SESAC, etc.
   - **Writer Splits**: How royalties are split
   - **Publisher**: Publishing information

6. **Submit Your Track**
   - Review all information
   - Click "Submit Music"
   - You'll receive a tracking ID

## ğŸ”§ Troubleshooting

### **Common Artist Issues**

**"No artists found" when submitting**
- Create an artist profile first at /artists
- Make sure you're logged in
- Check that your artist profile is active

**"Artist information not auto-filling"**
- Make sure you've selected an artist from the dropdown
- Verify your artist profile has complete information
- Try refreshing the page

**"File upload failed"**
- Check file size (must be under 150MB)
- Verify file format (MP3, WAV, M4A, FLAC)
- Check your internet connection

### **Common Admin Issues**

**"LibreTime connection failed"**
- Verify LibreTime server URL is correct
- Check that API key is valid
- Ensure backend can reach LibreTime server
- Test connection manually

**"ISRC key not saving"**
- Make sure key is exactly 3 characters
- Use uppercase letters only
- Check that you have admin permissions

**"Admin login failed"**
- Verify email and password are correct
- Check that admin account is active
- Try resetting password
